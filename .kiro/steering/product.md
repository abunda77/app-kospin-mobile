# Product Overview

**Sinara POS Expo App** is a React Native mobile application for Kospin Sinara Artha's Point of Sale system. The app provides a WebView-based interface to access the web POS application at `https://pos.kospinsinaraartha.co.id/m/catalog`.

## Key Features

- WebView integration with navigation controls (back, forward, refresh)
- Tab-based navigation with intuitive icons
- Automatic theme support (light/dark mode)
- Pull-to-refresh functionality
- WebView history management
- Exit confirmation dialog

## Target Platform

Primary target is Android mobile devices, with support for iOS and web platforms through Expo's cross-platform capabilities.
